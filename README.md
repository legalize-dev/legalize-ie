# legalize-ie

Irish legislation as Markdown files, version-controlled with Git.

## What's here

- **~4,000 Acts of the Oireachtas** (1922-2026) as individual Markdown files
- YAML frontmatter with title, identifier, publication date, status, and source URL
- Full text preserving official structure: Parts, Chapters, Sections
- Tables rendered as Markdown pipe tables
- Irish language characters (fada) correctly encoded

## File naming

Each file follows the pattern `ie/{identifier}.md`:

```
ie/IE-2024-act-1.md     # Policing, Security and Community Safety Act 2024
ie/IE-1997-act-39.md    # Taxes Consolidation Act 1997
ie/IE-2015-act-C34.md   # Thirty-fourth Amendment of the Constitution
```

## Source

All text comes from the [Irish Statute Book](https://www.irishstatutebook.ie/) (ISB), maintained by the Office of the Attorney General. Metadata from the [Oireachtas API](https://api.oireachtas.ie/).
